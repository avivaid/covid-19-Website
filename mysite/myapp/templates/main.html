A little bit about us

This websites show the data on the new covid-19 pandamic. The information is updated daily. The data is originally collected by the very hardworking team at Johns Hopkins University Center for Systems Science and Engineering (CSSE). The data is futher polished by a github user named jeffcore. The data is being pulled from his git repo. Links to both the git repos and the CSSE website can be found in the top bar at the home page.

This websites also provides a group chat services for its users but you must be logged in to use the group chat
